NaN

JavaScript term for "Not A Number", but is still considered a number.

Strings

Strings are a sequence of characters, found within quotation marks in code variables. In JavaScript, they are often considered variable-type objects.


Booleans

Booleans are a type of data point that exist as true or false, defined by their variable or certain conditions. For instance, a variable boolean could be set to true or false by default, or it can be determined by certain methods and conditions such as "if" statements in code.

Alerts

Alerts are a type of dialog box in JavaScript typically used to issue a warning to the user in most circumstances. However, in other cases, they can be programmed to tell the user any type of message in the dialog box.
